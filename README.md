# Sentiment-Analysis

### Workflow ###

* Run ```data_preprocessing_pos.ipynb``` and ```data_preprocessing_neg.ipynb``` to generate positive.npy and negative.npy.
* Then run the model in ```sentiment_analysis.ipynb``` to generate the predictions.
